1. Membuat Akun dan Repository GitHub

Buat akun GitHub Melalui website https://github.com

b. Clik “Creat respository”

c. Kemudian isi beberapa nama atau filed terserah kalian,dan juga isi beberapa filed
di dalam kolom “Description”.Karena hal ini harus diajukan

d. Untuk mengecek hasil dari repositorinya ,kalian bisa buka nama akun yang
telah kalian buat.Jika sudah benar maka akan sama seperti pada gambar dibawah ini
